The most up-to-date information for PhysUtil is located at http://code.google.com/p/python-physutil/wiki/PhysUtil

Also, check out the examples that were included in this download.

To use: 
Place physutil.py in the same directory as the VPython code you are working on. You can include all the classes simply by typing:
from physutil import *
You can also choose to import only a subset:
from physutil import MotionMapN, PhysGraph
